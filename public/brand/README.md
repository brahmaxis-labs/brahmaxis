## Logo System

- Horizontal logo, dark text: `logo-horizontal-dark.svg`, `logo-horizontal-dark.png`
- Horizontal logo, light text: `logo-horizontal-light.svg`, `logo-horizontal-light.png`
- Website header logo, dark background: `logo-header-dark.png`
- Vertical logo, dark text: `logo-vertical-dark.svg`, `logo-vertical-dark.png`
- Vertical logo, light text: `logo-vertical-light.svg`, `logo-vertical-light.png`
- Icon only, dark mark: `icon-only-dark.svg`, `icon-only-dark.png`
- Icon only, light mark: `icon-only-light.svg`, `icon-only-light.png`
- Favicon-ready icon on light background: `icon-only-dark-on-light.svg`
- Favicon-ready icon on dark background: `icon-only-light-on-dark.svg`

Icon SVG files are raster-backed wrappers from the approved icon reference image. Replace them with native vector artwork when final source SVG is available.
